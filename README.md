# Customer Shopping Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using Python, SQL Server, and Power BI. The objective is to discover purchasing patterns, customer segments, revenue trends, and business insights that can support decision-making.

## Tools Used

* Python
* Pandas
* NumPy
* SQL Server
* SQLAlchemy
* Power BI

## Project Workflow

### 1. Data Cleaning (Python)

* Renamed columns
* Standardized column names
* Handled missing review ratings using category median
* Removed duplicate records

### 2. Feature Engineering

* Created customer age groups
* Converted purchase frequency categories into numerical values

### 3. Database Integration

* Exported cleaned data to SQL Server using SQLAlchemy

### 4. SQL Analysis

Business questions answered:

* Revenue by gender
* Top-rated products
* Subscriber vs non-subscriber spending
* Discount effectiveness
* Customer segmentation
* Revenue contribution by age group

### 5. Power BI Dashboard

Dashboard includes:

* Total Revenue KPI
* Total Customers KPI
* Average Rating KPI
* Number of Cities KPI
* Revenue by Category
* Revenue by Age Group
* Top Revenue Cities
* Subscription Analysis
* Payment Method Distribution
